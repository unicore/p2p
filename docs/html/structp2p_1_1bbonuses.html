Detailed Description

Таблица резервов контракта для выплат бонусов в реферальную сеть

Table: bbonuses
Contract: _me
Scope: _me

Таблица пополняется переводом на аккаунт контракта с указаним в поле memo аккаунта продавца, который будет использовать распределение на сеть покупателя. Распределение срабатывает в момент завершения сделки, увеличивая значение в поле distributed согласно курсу распределения disctribution_rate.
