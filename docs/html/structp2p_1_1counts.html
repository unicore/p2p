Detailed Description

Таблица счётчиков ордеров

CONTRACT = _me, SCOPE = _me, TABLE = counts Используется для хранения счётчика ордеров с ключом totalorders. При создании нового ордера, счётчик увеличивается на 1. При завершении или удалении ордера, счётчик не изменяется.
