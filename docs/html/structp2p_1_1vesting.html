Detailed Description

Таблица вестинг-балансов пользователей

CONTRACT = _me, SCOPE = owner, TABLE = vesting

Пополняется контрактом в случае, если ключ _ENABLE_VESTING = TRUE на количество секунд в _VESTING_SECONDS, срабатывает только для аккаунта продавца _CORE_SALE_ACCOUNT. Позволяет заморозить покупку токенов у компании на указанное количество секунд.
